# Сайт рецептов

Лабораторная работа по Django. Платформа для публикации кулинарных рецептов.

## Установка

1. Создать виртуальное окружение и активировать.
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить PostgreSQL.
4. Выполнить миграции: `python manage.py migrate`
5. Запустить: `python manage.py runserver`
